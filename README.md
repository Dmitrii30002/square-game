# square-game
В данной игре необходимо найти такой же квадратик по раскраске, что и в примере. 
### Режимы игры 
Есть режима игры: 3x3, 4x4, 5x5, 6x6 - от режима зависит размер квадратика и количество вариантов ответа.
### Режимы сложности

Классический режим
 - Квадратики упорядочены
 - Один из вариантов повёрнут на 0, 90, 180 или 270 градусов
 - Нажимая на пример, можно поворачивать его по часовой стрелке на 90 градусов
 - Для выбора ответа необходимо нажать левой кнопкой мыши на вариант

Нестандартный режим
 - Квадратики упорядочены
 - Все варианты повёрнуты на произвольный градус
 - Нажимая на пример и двигая мышь, можно поворачивать его
 - Для выбора ответа необходимо нажать левой кнопкой мыши на вариант

Безумный режим
 - Квадратики неупорядочены
 - Все варианты повёрнуты на произвольный градус
 - Нажимая на пример и двигая мышь, можно поворачивать его
 - Для выбора ответа необходимо нажать правой кнопкой мыши на вариант
 - Квадратики-варианты можно передвигать зажимая мышь и отпуская
